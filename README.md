R-Code and data for a paper titled "K-Economy or Not? Evidence From a Payments Survey"

by Aditi Routh and Oz Shy

---------------------------------------

Instructions:

Download the R-code file: "k_spend_2026_mm_dd.R" and the 10 data files: "dcpc-YEAR-indlevel-public.RDS" and "dcpc-YEAR-tranlevel-public.RDS" into the same folder on your computer.

Run the ENTIRE code first (don't forget to change the working directory).

If you are using RStudio, after you run the ENTIRE code first, you can jump to the beginning and end of each table and figure by clicking the list of contents on the left-lower corner.
